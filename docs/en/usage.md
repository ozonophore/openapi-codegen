## Usage

> **Moved:** full CLI documentation is in **[cli-reference.md](cli-reference.md)**.
> Quick start: **[getting-started.md](getting-started.md)**.

The CLI binary is **`openapi-codegen-cli`** (not `openapi`).

```bash
npx openapi-codegen-cli generate --input ./spec.json --output ./dist
npx openapi-codegen-cli check-config
npx openapi-codegen-cli generate --openapi-config ./openapi.config.json
```

See [cli-reference.md](cli-reference.md) for all seven commands, flags, and the `check-config` executor warning table.
