## Использование

> **Перенесено:** полная документация CLI — **[cli-reference.md](cli-reference.md)**.
> Быстрый старт: **[getting-started.md](getting-started.md)**.

CLI-бинарник — **`openapi-codegen-cli`** (не `openapi`).

```bash
npx openapi-codegen-cli generate --input ./spec.json --output ./dist
npx openapi-codegen-cli check-config
npx openapi-codegen-cli generate --openapi-config ./openapi.config.json
```

См. [cli-reference.md](cli-reference.md) — все команды, флаги и таблица warnings `check-config`.
